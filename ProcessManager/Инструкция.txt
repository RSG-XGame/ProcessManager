Требования
.Net Framework 4.7.2
.Net Core 3.1

1. Установка службы
Чтобы начать установку поместите файлы "install.cmd" и "uninstall.cmd" рядом с программой.
Затем, для установки службы запустите от имени администратора файл "install.cmd" и дождитесь выполнения установки.
Для удаления выполните файл "uninstall.cmd" от имени администратора.

2. Команды программы
Для получения списка команд вызовите:
ProcessManager.exe --commandType=GetProcesses

Для остановки процесса:
ProcessManager.exe --commandType=Start --pathExe=<Путь к файлу программы> [--arguments=<аргументы запуска>]
--arguments - аргументы запуска являются не обязательными параметрами

Для перезапуска процесса:
ProcessManager.exe --commandType=Stop [--processName=<имя процесса>] [--processId=<id процесса>] [--pathExe=<путь к исполняемому файлу>]
Для выполнения команды остановки процесса достаточко указать только один из параметров.
[--processName=<имя процесса>] 
[--processId=<id процесса>] 
[--pathExe=<путь к исполняемому файлу>]

Для запуска процесса:
ProcessManager.exe --commandType=Restart [--processName=<имя процесса>] [--processId=<id процесса>] [--pathExe=<путь к исполняемому файлу>] [--arguments=<аргументы запуска>]
Для перезапуска достаточно указать только один из параметров.
Дополнительно можно указать аргументы, с которыми программа должна будет быть перезапущена
